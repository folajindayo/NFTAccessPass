# NFT Access Pass

A decentralized application (dApp) that gates content access based on NFT ownership.

## Features

- **NFT Verification**: Verify ownership of specific NFTs (ERC-721/ERC-1155).
- **Access Control**: Gate specific routes or components based on token holdings.
- **Wallet Connection**: Integrated with Reown AppKit/Wagmi for seamless wallet connection.

## Tech Stack

- **Frontend**: Next.js, React, TailwindCSS
- **Blockchain Interaction**: Viem, Wagmi
- **Smart Contracts**: Hardhat, OpenZeppelin

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) in your browser.
